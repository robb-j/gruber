# Browser

> EXPERIMENTAL

This module is the entrypoint for using Gruber in the browser. It's a slimmed down export of the modules that might be useful client-side.

### Install

Gruber is available through [NPM](https://www.npmjs.com/package/gruber)

```bash
npm install gruber
```
